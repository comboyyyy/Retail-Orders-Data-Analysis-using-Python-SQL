# Retail Orders Data Analysis Project

An end-to-end data analysis project utilizing Python for data cleaning/transformation and SQL for deep business insights.

## Project Structure
* `notebooks/orders.ipynb`: Python script handling data ingestion, data cleaning (handling nulls), formatting column names, and deriving performance metrics (Discount, Sale Price, Profit).
* `sql_scripts/analysis_queries.sql`: Advanced SQL queries addressing key business questions.

## Business Questions Answered
1. Top 10 highest revenue-generating products.
2. Top 5 highest selling products in each region.
3. Month-over-month growth comparison for 2022 and 2023.
4. Peak sales months for each product category.
5. Sub-category with the highest profit growth in 2023 vs 2022.

## Tech Stack
* **Python** (Pandas, NumPy)
* **SQL** (Common Table Expressions, Window Functions, Aggregations)
